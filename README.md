# 🛒 Clon MercadoLibre

Proyecto académico que replica la interfaz y funcionalidades básicas de MercadoLibre, desarrollado como parte del quinto semestre de Ingeniería de Software.

## 📋 Descripción

Este proyecto es un clon educativo de MercadoLibre que incluye:
- Barra de navegación responsiva
- Sistema de búsqueda de productos
- Catálogo de productos destacados
- Portafolio del equipo de desarrollo
- Footer informativo con enlaces

## 💻 Instalación y Uso

1. Clona el repositorio:
```bash
git clone [URL_DEL_REPOSITORIO]
```

2. Navega al directorio del proyecto:
```bash
cd mercadolibre-clone
```

3. Abre el archivo `index.html` en tu navegador preferido

No se requiere instalación de dependencias adicionales.
